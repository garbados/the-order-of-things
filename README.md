# The Order of Things

An epic poem in 78 parts, inspired by [Smith-Waite Tarot](https://en.wikipedia.org/wiki/Rider%E2%80%93Waite_Tarot) and [*The Epic of Gilgamesh*](https://en.wikipedia.org/wiki/Epic_of_Gilgamesh).

Check it out [online](https://garbados.github.io/the-order-of-things), where you can also draw Tarot spreads.
